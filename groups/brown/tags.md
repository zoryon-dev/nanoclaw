# Vocabulário de tags — Links — Biblioteca

Leia antes de taggear. Reuse uma tag existente sempre que servir; só crie tag nova quando
nada aqui encaixa, e então acrescente a nova abaixo. Tags curtas, minúsculas, pt-br ou termo
técnico comum em inglês.

## Tags em uso
- `asr` — reconhecimento de fala / speech recognition
- `speech-to-text` — transcrição de áudio em texto
- `openai` — produtos/repos da OpenAI
- `python` — projeto/ferramenta em Python
- `machine-learning` — ML / modelos de aprendizado
