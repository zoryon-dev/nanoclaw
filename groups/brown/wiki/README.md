# Wiki do Brown

Base de conhecimento própria do bibliotecário de links (Brown). Destilo aqui o que os
links salvos ensinam: conceitos, entidades (ferramentas/projetos/pessoas) e tópicos.
Um arquivo por nota, gerenciado pela skill de wiki. Não é histórico de conversa nem
substitui o Notion (`Links — Biblioteca`) — o Notion é o acervo; a wiki é o que aprendi.

Para contexto cruzado consulto, somente leitura: `/workspace/extra/wikis/{zory,caio,lobby}`.
