# Biblioteca de cues

Cues organizados por exercício e por foco atencional (interno ou externo).
Use foco interno para alunos aprendendo o movimento. Use foco externo quando
o aluno já domina e está performando.

Regra geral da literatura: foco externo melhora performance e retenção do
aprendizado motor. Foco interno acelera aquisição inicial em iniciantes.
Use o certo no momento certo.

## Como escolher cue

1. **Identifique o erro técnico mais crítico** que aluno está cometendo
2. **Escolha UM cue que ataca esse erro**, não vários
3. **Use linguagem que faz sentido para o aluno** (técnica vs cotidiana)
4. **Observe se o cue funcionou** (se não funcionou, troca por outro)

Não dê três cues seguidos. O cérebro do aluno em meio de exercício processa um
foco. Mais que um confunde.

## Squat (agachamento)

### Erros mais comuns
- Joelhos colapsando para dentro (valgo dinâmico)
- Tronco caindo para frente excessivamente
- Não atingindo paralela
- Perda de neutralidade de coluna na descida
- Calcanhares saindo do chão

### Cues por erro

**Joelho valgo (caindo pra dentro)**:
- Interno: "empurre os joelhos para fora, na direção dos dedinhos"
- Externo: "afaste o chão com os pés, como se quisesse abrir o solo"

**Tronco caindo pra frente**:
- Interno: "mantenha o peito aberto, como mostrando o logo da camiseta"
- Externo: "mantenha o peito apontando para o horizonte"

**Não atinge paralela**:
- Interno: "sente o quadril descer até quebrar a paralela com o chão"
- Externo: "sente entre os calcanhares, como se fosse uma cadeira atrás"

**Coluna perde neutralidade**:
- Interno: "ative o core como se fosse levar um soco na barriga"
- Externo: "mantenha a barra firme nas costas durante todo o movimento"

**Calcanhares saindo**:
- Interno: "pressione os calcanhares contra o chão"
- Externo: "empurre o chão para baixo, distribuindo peso entre calcanhar e meio do pé"

## Deadlift (levantamento terra)

### Erros mais comuns
- Coluna em flexão (lombar curvada)
- Quadril subindo antes do tronco
- Barra longe do corpo
- Hiperextensão de coluna no topo
- Joelho colapsando

### Cues por erro

**Coluna curvada**:
- Interno: "imagine um cabo de vassoura nas costas, mantenha contato em três pontos: cabeça, costas, sacro"
- Externo: "mostre o número da camisa para o teto antes de puxar"

**Quadril subindo primeiro**:
- Interno: "empurre o chão para baixo com os pés enquanto sobe o peito"
- Externo: "leve barra e peito para o teto ao mesmo tempo"

**Barra longe do corpo**:
- Interno: "raspe a canela enquanto sobe"
- Externo: "leve a barra colada ao corpo como se fosse imã"

**Hiperextensão no topo**:
- Interno: "termine com pélvis neutra, glúteo apertado, não jogue quadril pra frente"
- Externo: "fique de pé como se fosse uma régua, não curve a coluna pra trás"

## Bench press (supino)

### Erros mais comuns
- Cotovelos abrindo a 90 graus (flare excessivo)
- Glúteos saindo do banco
- Barra batendo no pescoço ou no abdômen
- Punhos quebrando
- Não tocando o peito

### Cues por erro

**Cotovelos abrindo**:
- Interno: "puxe os cotovelos para o seu corpo a uns 60-75 graus"
- Externo: "faça os cotovelos apontarem mais para o quadril que para o teto"

**Glúteo saindo do banco**:
- Interno: "mantenha contato do glúteo no banco durante toda a série"
- Externo: "cole a bunda no banco como se ela fizesse parte"

**Barra fora de posição**:
- Interno: "desça a barra na linha do peitoral, pouco abaixo do mamilo"
- Externo: "leve a barra para baixo até tocar o peito na altura do esterno"

**Punhos quebrando**:
- Interno: "mantenha o punho reto, alinhado com o antebraço"
- Externo: "esmague o cabo da barra, segure como se quisesse quebrar"

## Overhead press (desenvolvimento militar)

### Erros mais comuns
- Hiperlordose lombar para compensar
- Barra à frente da linha dos ombros no topo
- Movimento descontrolado na descida
- Cabeça não passa pela barra (alinhamento)

### Cues por erro

**Lordose excessiva**:
- Interno: "aperte o glúteo e o core, mantenha a costela baixa"
- Externo: "fique reto como uma régua, sem curvar a coluna"

**Barra à frente**:
- Interno: "termine com a barra acima do meio do crânio"
- Externo: "coloque a cabeça embaixo da barra no topo, atravesse a janela"

**Descida descontrolada**:
- Interno: "controle a descida em 2-3 segundos, não solte"
- Externo: "guie a barra como se estivesse colocando uma pintura no batente"

## Row (remada)

### Erros mais comuns
- Cotovelo abrindo lateralmente
- Tronco subindo (uso de momentum)
- Não retraindo escápula
- Curvatura lombar

### Cues por erro

**Cotovelo abrindo**:
- Interno: "puxe o cotovelo para trás, próximo ao corpo"
- Externo: "leve o cotovelo para a parede atrás de você"

**Uso de momentum**:
- Interno: "isole o movimento nas costas, não use a coluna pra puxar"
- Externo: "mantenha o tronco como uma tábua durante todo o movimento"

**Escápula não retrai**:
- Interno: "aperte as escápulas como se segurando uma moeda no meio das costas"
- Externo: "leve os ombros para os bolsos de trás da calça"

## Pull-up / chin-up

### Erros mais comuns
- Não iniciar com escápula (puxa só com braço)
- Não passar queixo da barra
- Balanço excessivo (kipping involuntário)
- Não desce completamente

### Cues por erro

**Não inicia com escápula**:
- Interno: "antes de dobrar o cotovelo, baixe as escápulas como se quisesse colocar elas no bolso"
- Externo: "afaste a barra do teto antes de subir"

**Não passa queixo**:
- Interno: "puxe o peito em direção à barra, não só o queixo"
- Externo: "leve as clavículas para a barra"

**Balanço excessivo**:
- Interno: "ative o core e o glúteo, mantenha o corpo rígido"
- Externo: "fique como uma vela acesa, sem oscilar"

## Lunge (afundo)

### Erros mais comuns
- Joelho da frente passando da ponta do pé
- Joelho de trás batendo no chão
- Tronco caindo pra frente
- Perda de equilíbrio lateral

### Cues por erro

**Joelho ultrapassa o pé**:
- Interno: "sente o quadril descer reto, mantendo o joelho atrás dos dedos do pé"
- Externo: "desça reto como um pistão, não pra frente"

**Tronco cai**:
- Interno: "mantenha o tronco ereto, peito para frente"
- Externo: "fique alto, como se tivesse um chapéu para equilibrar na cabeça"

## Hip thrust / glute bridge

### Erros mais comuns
- Hiperextensão lombar no topo (em vez de extensão de quadril)
- Não chegar a extensão completa
- Joelho caindo pra dentro
- Pés mal posicionados

### Cues por erro

**Hiperextensão lombar**:
- Interno: "termine com glúteo apertado, sem jogar quadril pra cima do nível"
- Externo: "fique paralelo ao chão no topo, como uma mesa de café"

**Não chega completo**:
- Interno: "empurre o quadril o mais alto que conseguir sem dobrar a coluna"
- Externo: "leve o quadril para o teto"

**Joelho caindo pra dentro**:
- Interno: "empurre os joelhos para fora ativamente"
- Externo: "abra os joelhos como se tivesse um elástico apertando eles para dentro"

## Push-up (flexão)

### Erros mais comuns
- Quadril caindo (perda de prancha)
- Cotovelos abrindo a 90 graus
- Cabeça caindo
- Não desce completamente

### Cues por erro

**Quadril caindo**:
- Interno: "ative o core e o glúteo, mantenha o corpo reto"
- Externo: "fique como uma tábua de madeira do calcanhar à cabeça"

**Cotovelos abrindo**:
- Interno: "mantenha cotovelos a 45 graus do corpo, não 90"
- Externo: "faça cotovelos apontarem para trás, não para os lados"

## Plank (prancha)

### Erros mais comuns
- Quadril caindo
- Quadril subindo
- Ombros encolhidos
- Cabeça caindo

### Cues por erro

**Quadril desalinhado**:
- Interno: "aperte glúteo e abdômen para manter pélvis neutra"
- Externo: "fique como uma tábua reta do calcanhar à cabeça"

**Ombros encolhidos**:
- Interno: "empurre o chão para longe, mantendo escápulas baixas"
- Externo: "afaste o chão como se quisesse empurrá-lo"

## Cues gerais por padrão motor

Quando o exercício específico não está nesta lista, use os cues genéricos
por padrão motor:

**Padrão push (qualquer empurrada)**:
- Interno: "estabilize a escápula primeiro, depois empurre"
- Externo: "afaste [o objeto] de você"

**Padrão pull (qualquer puxada)**:
- Interno: "puxe com as costas, não com o braço"
- Externo: "leve [o objeto] em direção ao corpo"

**Padrão hinge (dobra de quadril)**:
- Interno: "dobre o quadril, não a coluna"
- Externo: "leve o quadril para trás, como se fechasse uma porta com a bunda"

**Padrão squat (agachamento)**:
- Interno: "desça com peito alto e joelhos alinhados"
- Externo: "sente entre os calcanhares"

**Padrão lunge**:
- Interno: "desça reto, joelho atrás do pé"
- Externo: "desça como um pistão"
