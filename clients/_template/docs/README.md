# Base de Conhecimento

Adicione documentos aqui para que os agentes possam consultá-los.

## Como Usar

1. Crie arquivos `.md` com informações da empresa
2. Referencie os documentos na configuração de cada agente
3. Os agentes terão acesso somente aos documentos listados

## Exemplos de Documentos

- `faq.md` - Perguntas frequentes
- `politicas.md` - Políticas da empresa
- `produtos.md` - Catálogo de produtos/serviços
- `precos.md` - Tabela de preços
- `processos.md` - Processos internos
