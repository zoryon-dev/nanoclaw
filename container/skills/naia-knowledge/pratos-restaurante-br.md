# Pratos de Restaurante Brasileiro — macros estimados

> Catálogo de pratos típicos servidos em restaurantes/bufês/delivery brasileiros, com estimativa de macros por porção.
> **Importante:** valores são estimativas baseadas em receitas tradicionais. Variação real pode ser ±20%.
> Use junto com `medidas-caseiras.md` para porcionar.

## Como ler o ranking

Cada prato tem 3 informações-chave para a Naia decidir rápido:

- **🥇/🥈/🥉/❌** — ranking padrão da Naia (top opção / boa / aceitável com ajuste / evitar)
- **Por quê** — motivo direto da classificação
- **Como pedir** — ajuste tático para melhorar o encaixe no plano High Carb

## PROTOCOLO DA NAIA EM CARDÁPIO COM FOTO

Quando Jonas mandar foto de cardápio:

```
1. Identifica o tipo de restaurante (PF/churrasco/japonês/etc)
2. Lista todos os pratos lidos
3. Para cada um, busca neste arquivo o ranking + macros
4. Filtra por: tem proteína identificável? carbo aceitável? evita fritura/cremoso?
5. Apresenta TOP 3 + 1 evitar:
   - 🥇 [prato] — kcal/PTN — Por quê encaixa. Como pedir: [ajuste]
   - 🥈 [prato] — kcal/PTN — boa segunda opção
   - 🥉 [prato] — aceitável se [condição]
   - ❌ Evite: [pior prato] — motivo
6. Lembra a regra do prato: ½ vegetais, ¼ proteína, ¼ carbo
```

## QUANDO O PRATO NÃO ESTÁ NA LISTA

1. Tenta encaixar em uma categoria similar (ex: "filé à milanesa" ≈ "bife à parmegiana" → ❌ fritura+queijo)
2. Decompõe em ingredientes-base e estima via TACO + queries CalorieNinjas
3. Sinaliza para Jonas: "Não tenho exato esse, estimei em ~X kcal / Y proteína. Para precisão maior, me passa o que vem no prato."
